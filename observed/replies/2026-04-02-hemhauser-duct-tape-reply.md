# Reply text

Thank you Daniel for this post. I found it very thought provoking.

It's crystallized something that I've been struggling to express for a while.

Project managers have a different attitude to bad news. They try to turn it into beneficial action.

And my favourite bits of being a project manager are when it "sings" - like a sailing ship that has all the sails set correctly and it's pointing in the right direction. It doesn't happen often, but when it does!!!
