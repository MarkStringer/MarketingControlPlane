This is the Knowledge Control Plane for marketing two products:

1. **Delivering the Impossible** — a book about software project management published by Springer
2. **You Can Write a Book** — a live, interactive Edinburgh Festival Fringe show

The show is an indirect marketing vehicle for the book: every audience member gets a branded notebook and pen, and starts writing during the show. The show's themes (writing, creativity, point of view) connect to the book's themes (seeing differently, project delivery) via a shared idea: there are no mirrors, only prisms.
